# User & Task Manager Application

This project consists of a full-stack application with a backend (Node.js/Express) and a frontend (React). Follow the steps below (STARTUP) to run the application locally.

---

## Prerequisites

Ensure you have the following installed:

- Node.js (version 14.x or higher)
- npm (Node Package Manager)

You can check if you have them installed by running:

```bash
node -v
npm -v

### **STARTUP** ###

- **Two Terminals**: 
  - One for the **backend** (`user-task-manager`) to run the server using `npm run dev`.
  - One for the **frontend** (`user-task-manager-frontend`) to run the React app using `npm start`.
  
- **Backend Command (`npm run dev`)**: 
  This should be defined in the `package.json` of the backend folder. If you're using `nodemon` for automatic server restarts, this command is typical.

- **Frontend Command (`npm start`)**:
  This is the default start command for React apps created with `create-react-app`.
